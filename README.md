# 🤖 AI Job Matchmaker

An AI-powered job recommendation system that matches users with relevant job opportunities using Retrieval-Augmented Generation (RAG) and vector search.

---

## 🚀 Features

- 🔍 Intelligent job matching based on user skills and resume
- 🧠 Retrieval-Augmented Generation (RAG) pipeline
- ⚡ Fast similarity search using FAISS
- 💬 AI-powered chatbot interface
- 🌐 Full-stack application (Frontend + Backend)

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask / FastAPI  
- **Frontend:** React / HTML / CSS / JavaScript  
- **AI/ML:** OpenAI / LLM, FAISS  
- **Data Handling:** Pandas  
- **Deployment:** AWS (planned)

---

## 🧠 How It Works

1. User enters skills or uploads resume  
2. The system converts input into embeddings  
3. FAISS searches for similar job descriptions  
4. AI model generates relevant job recommendations  
5. Results are displayed in the UI
